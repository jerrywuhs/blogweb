"又到周末了，周报还没写..."

"这个月的数据分析报告怎么办？"

用上这套模板，10分钟生成一份专业报告。

---

## 报告类模板清单

### 1. 日报生成

```
你是一个日报助手。
今日工作：
1. [任务1] - [完成度]
2. [任务2] - [完成度]
3. [任务3] - [完成度]

今日问题：
- [问题]

明日计划：
1. [计划1]
2. [计划2]

请生成简洁的日报（200字以内）。
```

### 2. 周报生成

```
你是一个周报助手。
本周工作记录：
1. [任务1] - 完成度：[百分比]
2. [任务2] - 完成度：[百分比]
3. [任务3] - 完成度：[百分比]

本周问题：
- [问题1]
- [问题2]

下周计划：
1. [计划1]
2. [计划2]

请按以下格式生成周报：
## 本周工作总结
[量化成果]

## 下周工作计划
[具体任务]

## 问题和挑战
[简要说明]
```

### 3. 月报生成

```
你是一个数据分析师。
本月关键数据：
- 销售额：[金额]，环比[增长/下降]%
- 客户数：[数量]，新增[数量]
- 转化率：[百分比]，环比[变化]
- 重点工作：[工作1、工作2...]

本月问题：
- [问题]

下月目标：
- [目标1]
- [目标2]

请生成月度工作报告。
```

### 4. 竞品分析报告

```
你是一个战略分析师。
竞品：[竞品名称]
分析维度：
1. 公司背景
2. 产品功能
3. 市场表现
4. 定价策略
5. 优劣势分析
6. 对我们的启示

请生成结构化分析报告。
```

## 使用技巧

1. **准备好基础数据**（不要让AI无中生有）
2. **明确报告目的**（汇报/决策/存档）
3. **指定输出格式**（表格/列表/段落）
4. **人工检查数据准确性**
