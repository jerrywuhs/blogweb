## 前言

掌握了基础模板，再学几个高级技巧，让你的AI输出质量提升一个档次！

---

## 高级技巧

### 1. 角色扮演（Persona）

让AI扮演特定角色，获得更专业的回答。

```
你是一个有10年经验的麦肯锡咨询顾问。
擅长结构化思维、金字塔原理。
请用咨询顾问的专业视角分析以下问题：
[问题描述]
```

**效果**：回答更专业、更有深度

### 2. 思维链（Chain of Thought）

让AI逐步推理，避免错误。

```
请按以下步骤分析这个问题：
步骤1：明确问题的核心是什么
步骤2：列出可能的因素
步骤3：分析每个因素的影响
步骤4：给出最终结论

问题：[你的问题]
```

**效果**：推理更严谨，答案更准确

### 3. 迭代优化

多轮对话不断改进。

```
第1轮：生成初稿
请帮我写一封[主题]的邮件。

第2轮：改进
请让语气更正式一些。

第3轮：精简
请把邮件压缩到200字以内。
```

**效果**：直到满意为止

### 4. Few-Shot示例

给AI看例子，它会模仿。

```
请模仿以下风格写一段话：
"产品是1，营销是0。没有好的产品，再多的营销也是徒劳。"

请用同样的风格写一段关于[主题]的话。
```

**效果**：风格更统一，质量更稳定

### 5. 限制输出格式

明确指定格式，避免格式混乱。

```
请用Markdown格式输出：
## 标题
### 要点1
- 细节
- 细节
### 要点2
- 细节
- 细节
```

**效果**：格式整齐，方便使用

---

## 最佳实践

### ✅ 应该做

1. **明确具体**：越具体越好
2. **提供背景**：给足上下文
3. **指定格式**：告诉AI要什么格式
4. **分步提问**：复杂问题拆成小问题
5. **检查结果**：AI也会犯错

### ❌ 不要做

1. **模糊不清**："帮我写点东西"
2. **期望读心**："按照之前的方式处理"
3. **没有约束**：不指定长度、格式
4. **一次问太多**：大任务拆开问
5. **不检查结果**：直接使用
