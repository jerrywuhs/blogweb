## 数据分析模板清单

### 1. Excel数据处理

```
你是一个Excel专家。
数据描述：[数据问题]
需求：
1. [需求1]
2. [需求2]
3. [需求3]

请给出具体的操作步骤或公式。
```

### 2. 图表建议

```
你是一个数据可视化专家。
数据类型：[类型]
展示目的：[目的]
数据关系：[对比/趋势/构成/分布]
目标受众：[受众]

请推荐合适的图表类型，并说明理由。
```

### 3. 数据洞察分析

```
你是一个数据分析师。
数据：
- [指标1]：[数值]
- [指标2]：[数值]
- [指标3]：[数值]

分析要求：
1. 描述性统计
2. 趋势分析
3. 异常值识别
4. 关键发现
5. 建议与结论
```

### 4. 漏斗分析

```
你是一个增长分析师。
漏斗步骤：
1. [步骤1]
2. [步骤2]
3. [步骤3]

转化数据：
- 步骤1→步骤2：[百分比]
- 步骤2→步骤3：[百分比]

请分析转化情况，找出优化点。
```

### 5. 数据报告生成

```
你是一个数据分析师。
数据描述：[数据背景]
分析维度：[维度]
关键指标：[指标]

请生成一份数据分析报告，包含：
1. 执行摘要
2. 数据概览
3. 关键发现
4. 建议与结论
```

---

## 使用技巧

1. **明确分析目的**（看趋势/找异常/做对比）
2. **提供清晰数据**（格式规范、字段明确）
3. **选择合适模板**（不同场景用不同模板）
4. **验证AI结果**（数据准确性需要核对）
