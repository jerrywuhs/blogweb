## 文案类模板清单

### 1. 小红书种草文案

```
你是一个小红书种草达人。
产品：[产品名称]
核心卖点：[卖点1、2、3]
使用场景：[场景]
价格：[价格]
目标用户：[人群]

要求：
- 标题吸引眼球（带emoji）
- 正文真实感强
- 开头抓人眼球
- 中间分享使用感受
- 结尾引导互动
- 适当添加话题标签
```

### 2. 朋友圈文案

```
你是一个社交媒体运营。
主题：[主题]
风格：[风格]
字数：50-100字
要素：emoji、痛点、解决方案、互动引导

请生成朋友圈文案。
```

### 3. 公众号文章

```
你是一个[领域]专家。
文章主题：[主题]
目标读者：[读者画像]
核心观点：[观点]
篇幅：[字数要求]

结构：
1. 开头：引入话题，引发共鸣
2. 背景：说明背景，建立认知
3. 主体：3-5个分论点
4. 案例：2-3个真实案例
5. 结论：总结观点，给出建议
```

### 4. 产品描述文案

```
你是一个电商文案专家。
产品：[产品名称]
核心功能：[功能]
产品卖点：[卖点]
使用场景：[场景]

要求：
- 突出核心卖点
- 解决用户痛点
- 语言简洁有力
- 引导购买决策
```

---

## 使用技巧

1. **明确平台特点**（小红书重种草，公众号重深度）
2. **了解目标用户**（年龄、性别、痛点）
3. **提供足够信息**（产品卖点、使用场景）
4. **生成后人工优化**（加入个人特色）
