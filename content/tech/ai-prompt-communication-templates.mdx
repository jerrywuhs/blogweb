## 沟通类模板清单

### 1. 请求上级支持

```
你是一个[职位]。
背景：[工作背景]
需要支持：[事项]
原因：[原因]
预期效果：[效果]
紧迫程度：[程度]

请帮我写一段简洁有力的向上级请求支持的沟通话术。
```

### 2. 汇报工作进展

```
你是一个职场沟通专家。
情况：[现状]
进展：[进度]
问题：[如有]
请求：[请求]

请帮我组织一段汇报话术，结构清晰、重点突出。
```

### 3. 跨部门协作请求

```
你是一个[职位A]。
需求：[需求内容]
原因：[原因]
时间要求：[时间]
协作方式：[方式]

请帮我写一段跨部门协作请求的话术，友好但明确。
```

### 4. 反馈问题

```
你是一个[职位]。
问题：[描述]
影响：[影响]
原因：[原因]
建议：[建议]

请帮我组织一段反馈问题的话术，客观、有建设性。
```

### 5. 项目进度同步

```
你是一个项目经理。
项目：[项目名称]
当前进度：[进度]
下步计划：[计划]
需要配合：[配合事项]

请帮我生成项目进度同步消息。
```

---

## 使用技巧

1. **提前想清楚沟通目的**
2. **准备好关键信息**（数据、时间、人物）
3. **选择合适的语气**（正式/友好/简洁）
4. **生成后读一遍，确保无误**
