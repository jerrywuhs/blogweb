## 前言

"明明问的一样，为什么他的回答比我的好？"

很可能，你踩了这些坑...

---

## 5大常见误区

### ❌ 误区1：太模糊

**错误示范**：
> "帮我写一封邮件"

**问题**：什么邮件？给谁？什么目的？

**正确做法**：
```
你是一个商务秘书。
背景：感谢合作方完成项目交付。
收件人：项目经理张总。
目的：表达感谢并邀请进一步合作。
要求：专业简洁，200字左右。
```

---

### ❌ 误区2：缺少约束

**错误示范**：
> "给我一些建议"

**问题**：多少条？什么方向？

**正确做法**：
```
请给出5条关于[主题]的建议。
每条不超过50字。
按重要性排序。
```

---

### ❌ 误区3：期望读心

**错误示范**：
> "按照之前的方式处理"

**问题**：AI不记得"之前"是什么

**正确做法**：
```
背景：之前我们用的是[方法A]。
新任务：[新任务描述]。
要求：沿用[方法A]的风格处理新任务。
```

---

### ❌ 误区4：一次问太多

**错误示范**：
> "帮我分析这个市场，还要写一份报告，做一个PPT，再想几个营销方案"

**问题**：任务太杂，AI无法专注

**正确做法**：
```
任务1：分析市场现状
任务2：生成报告（等任务1完成后）
任务3：设计PPT（等任务2完成后）
任务4：营销方案（全部完成后）
```

---

### ❌ 误区5：不检查结果

**错误示范**：直接复制AI的回答使用

**问题**：AI也会犯错

**正确做法**：
1. 检查事实准确性
2. 检查逻辑完整性
3. 检查格式正确性
4. 根据需要修改

---

## 避坑清单

- **太模糊** - 表现：问题笼统 - 解决方法：5W1H分析法
- **缺少约束** - 表现：范围过大 - 解决方法：指定数量/格式
- **期望读心** - 表现：依赖上下文 - 解决方法：每次都提供背景
- **一次问太多** - 表现：任务过多 - 解决方法：拆分成小任务
- **不检查** - 表现：直接使用 - 解决方法：人工复核

---

## 实用工具

### 5W1H提问法

- **What** - 做什么
- **Why** - 为什么
- **Who** - 谁来做
- **When** - 什么时候
- **Where** - 在哪里
- **How** - 怎么做
