## 邮件类模板清单

### 1. 商务请求邮件

```
你是一个资深商务秘书。
背景：我们正在与[公司名]洽谈[项目名]合作。
任务：写一封请求提供资料的邮件。
要点：
1. 明确说明需要什么资料
2. 说明资料用途
3. 设定合理截止时间
4. 表达感谢
语气：专业、简洁、有礼貌
输出：完整的商务邮件
```

### 2. 邮件感谢

```
你是一个项目经理。
背景：合作方[公司名]在[项目]中提供了重要支持。
任务：写一封感谢邮件。
要求：
1. 具体说明帮助内容
2. 强调合作价值
3. 展望未来合作
输出：完整的感谢邮件
```

### 3. 邮件道歉

```
你是一个客服主管。
背景：因为[原因]导致[问题]，需要向客户道歉。
任务：写一封道歉邮件。
要求：
1. 诚恳道歉，态度真诚
2. 说明原因（简要）
3. 提出补救措施
4. 承诺改进
输出：完整的道歉邮件
```

### 4. 周报邮件

```
你是一个周报助手。
背景：本周工作总结。
本周完成：
1. [任务1]
2. [任务2]
3. [任务3]

本周问题：
- [问题]

下周计划：
1. [计划1]
2. [计划2]

请按格式生成周报邮件。
```
